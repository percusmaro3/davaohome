Accessibility Advice

Toolbar

Press ALT+F10 go to toolbar. Next button is TAB before button is go to SHIFT+TAB. Press ENTER, the function is activate and move the focus to Writing category. Press ESC, do not activate any function and move the focus to Writing category.

Go back

Press ALT+ go next to Smart Editor Press ALT+, go back to before Smart Editor

  • SHIFT+TAB Increase Indent
  • CTRL+U Underline
  • CTRL+F Find
  • CTRL+I Italic Font
  • CTRL+H Replace
  • CTRL+D Strikethrough
  • CTRL+K Link
  • TAB Decrease Indent
